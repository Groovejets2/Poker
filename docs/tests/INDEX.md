## How to Add Test Documentation

1. **New test results** (e.g. clinical test run output): create `YYYY-MM-DD_test-results-description_v1.0.md`
2. **Bug reports** (from clinical testing): create `YYYY-MM-DD_bug-report-description_v1.0.md`
3. **Updated test counts** (new unit tests added): update `TEST-PLAN.md` section 3 and bump the version
4. Add any new documents to this INDEX
